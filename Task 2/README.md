# Task 2 — Self-study: Heap (Data Structure) + Heap Sort (Algorithm)

## Chosen Topics
- **Data Structure:** Max Heap
- **Algorithm:** Heap Sort

## Why Heap & Heap Sort
- Heap is a complete binary tree structure commonly used for **priority queues** and **scheduling**.
- Heap operations (e.g., insert / extract-max) run in **O(log n)**.
- Heap sort runs in **O(n log n)** time with **O(1)** extra space (in-place variant).
