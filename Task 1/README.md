# Task 1 — Disaster Emergency Simulation System (Fire + Rescue + Weather Visualization)

## Background & Goal
This project simulates an urban fire emergency scenario in Python. Using Object-Oriented Programming (OOP), it models interactions among fire spread, smoke impact, rescue dispatch, traffic response, and weather effects. A 2D visualization (planned with pygame) will be used to present the simulation process clearly.

## Pre-submission Scope
This pre-submission provides:
- A preliminary OOP class scaffold and repository structure
- Placeholders for future implementations (fire spread rules, dispatch strategy, traffic logic, weather modifiers, and visualization)

## Planned Core Classes (Draft)
- `City`: manages the grid map, buildings, vehicles, and rescue units
- `Building` / `FlammableBuilding`: demonstrates inheritance for different building types
- `Fire`: manages fire states and spread logic
- `Vehicle` / `EmergencyUnit` / `FireTruck`: demonstrates inheritance and polymorphism
- `Weather`: modifies simulation parameters (e.g., spread speed, smoke range)
